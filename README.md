🎓 Student Registration System

A modern Student Registration Web Application built using HTML, CSS, and JavaScript.
The system allows users to register student details, store data locally using LocalStorage, and search registered students instantly.

📌 Project Overview

The Student Registration System collects student information including name, roll number, department, gender, and contact details.
All registered data is saved in the browser using LocalStorage, allowing records to remain even after refreshing the page.

This project demonstrates basic front-end data management without using a backend database.

✨ Features

📝 Student Registration Form

💾 LocalStorage data saving

🔍 Real-time student search

📋 Automatic student table display

🎨 Modern glass-style user interface

📱 Responsive layout

⚡ Works completely offline
